# 목록 스타일

## list-style-type

- 순서가 없는 경우: 불릿모양 지정 가능
- 순서가 있는 경우: 번호스타일 지정 가능

  1. disc
    안을 채운 원 불릿  ●
  
  2. circle
    빈 원 불릿  ○

  3. sqare
    안을 채운 사각형  ■

  4. decimal
    1부터 시작하는 10진수
  
  5. decimal-leading-zero
    앞 0부터 시작하는 10진수

  6. lower-roman
    로마숫자 소문자

  7. upper-roman
    로마숫자 대문자
  
  8. lower-alpha & lower-latin
    알파벳, 라틴어 소문자
  
  9. upper-alpha & upper-latin
    알파벳, 라틴어 대문자
  
  10. none
    불릿이나 숫자 제거
  
  ### 사용하기
  ```css
  .book1{
    list-style-type: none; /*순서가 없는 목록*/
  }

  
  ```
