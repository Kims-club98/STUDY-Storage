# 1. 파이썬 숫자
  - 파이썬에는 다음과 같은 3가지 유형이 있다
  - 숫자 변수는 값을 할당 시 생성된다.
    1.  int
    2. float
    3. complex
```py
x = 1 # int
y = 3.14 # float
z = 1j # complex
```

## int 정수
  - 소수점이 없는 양수 또는 음수이다
  - 길이제한이 없음

```py
# 예시
x = 1
y = 364156856468
z = 14265843
```

## float 실수
  - 양수, 음수가 가능함
  - *소수점 1자리 이상의 숫자*
```py
x = 1.56
y = 1.0
z = -3.556
```
  - 소수점 이하 10자리 거듭제곱은 "e"를 사용하여 표기 가능
```py
x = 35e3
y = 12E4
z = -87.7e100
```

## complex 복소수
  - 실수부(Real part)와 허수부(imaginary Part)가 존재하는 소수
  - 허수 부분을 "j"로 표현한다.

```py
x = 3+5j
y = 5j
z = -5j
```

# 2. 유형 변경하기
  - int(), float(), complex() 함수를 이용하여 유형을 변경할 수 있음

```py
x = 1    # int
y = 2.8  # float
z = 1j   # complex

# int -> float 로 유형 변환
a = float(x)

# float -> int 로 유형 변환
b = int(y)

# int -> complex로 유형 변환
c = complex(x)

print(a) # 1.0
print(b) # 2 (int 이므로 0.8은 사라짐)
print(c) # 1+0j
```
