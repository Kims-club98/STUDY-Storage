# 1. print()

- print는 텍스트를 출력하는 함수이다.
- 또한 print()는 원하는 만큼 여러 번 사용이 가능하다.(호출 시 텍스트는 새 줄에 출력된다.)

```py
print("Hello World!")
print("I am learning Python.")
print("It is awesome!")
# 각각 1줄씩 아래에 출력(3줄에 걸쳐서 출력된다.)
```

# 2. 텍스트 묶음: 따옴표("", '')

- 파이썬에서의 텍스트는 따옴표로 묶어야 한다(안 넣을 경우 오류 발생)
- 큰따옴표, 작은따옴표 둘 다 사용은 가능하나, 세트는 지켜줘야 한다( " ' 조합 불가)
```py
print(Hello World)
# 오류(텍스트문에 따옴표가 없으면 오류가 발생): SyntaxError: invalid syntax.
```

# 3. 줄바꿈 없이 인쇄하기
  - 한 줄에 여러 단어를 출력하고 싶으면 end="" 메개변수를 활용하면 된다.
```py
print("안녕하세요", end=" ") # 가독성을 위해 띄어쓰기를 하면 띄워진다.
print("반가워요")
# 출력: 안녕하세요반가워요
``
