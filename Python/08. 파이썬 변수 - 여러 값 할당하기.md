# 1. 여러 변수와 여러 값

- 파이썬은 한 줄로 여러 변수에 값을 할당할 수 있음

```py
x, y, z = 'Orange', 'Banana', 'Cherry' # 각 변수에 각 값이 들어감
print(x) # Orange
print(y) # Banana
print(z) # Cherry
```

# 2. 여러 변수에 1개의 값을 넣을 수 있다.

- 여러 변수에 동일한 값을 할당 가능하다.

```py
x = y = z = "오렌지" # 등호로 여러 변수에 1개의 값을 넣음
print(x) # 오렌지
print(y) # 오렌지
print(z) # 오렌지
```

# 3. 언페킹(unpacking)

- list에 값들이 있으면 -> tuple은 값을 변수로 추출 가능함

```py
fruits = ["apple", "banana", "cherry"]
x, y, z = fruits # list -> 변수 지정 시 tuple로 변수추출가능
print(x) # apple
print(y) # banana
print(z) # cherry
```
