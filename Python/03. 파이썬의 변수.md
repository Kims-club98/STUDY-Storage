# 1. 파이썬의 변수

- 파이썬의 변수는 **값을 할당할 경우** 생성
- 파이썬은 변수를 선언하는 명령어가 원칙적으로 없다

```py
# x와 y에 각각의 변수를 선언하고, print해줌
x=5
y="Hello World"
print(x,y)
```

# 2. 파이썬의 주석
  - 파이썬의 주석은 쉼표(#) 이며, 이는 나머지 줄을 주석으로 렌더링하는 역할을 한다.

# 3. Statement(문장)
  - Statement는 컴퓨터가 실행해야 하는 명령어의 목록이다.
```py
# python is fun! 이라는 문장을 출력해줌
print("python is fun!")
```

# 4. 여러 문장
  - 문장이 여러 개인 경우 작성된 순서되로 **위에서 아래로** 하나씩 실행된다.
```py
# 다음의 경우 Hello World -> Have a good day -> Learning Python is fun 순서로 출력된다.
print("Hello World!")
print("Have a good day.")
print("Learning Python is fun!")
```

# (선택) 세미콜론 (;)
  - 문장을 구분해주므로 1줄에 여러 문장을 작성해줄 수 있으나, 가독성이 떨어지므로 잘 사용하지 않는다.
```py
print("Hello"); print("How are you?"); print("Bye bye!")
# 출력은 3줄로 나눠서 출력된다.
# 오류(쉼표가 없으면 오류발생): SyntaxError: invalid syntax
```
