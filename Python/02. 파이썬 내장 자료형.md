# python 내장 자료형 종류

## 숫자

1. int (정수)
   100 -10 , 0, 1...

2. float (실수)
   -3.14, 0.1, 2.6e3

3. complex (복소수)
   -3-4j, 0j, 3+4j

## 시퀸스

1. str (문자열)
   'love' '안녕'

2. list (리스트)
   [] 요소의 여러 값

3. tuple
   () 요소의 변경 가능하지 않은 여러 값

## 기타

1. bool (불린)

2. dict (사전)
   키 - 값 구분
   {1: '남자'}

3. set (집합)
   {} 수학적 집한 연산 가능

4. frozenset (집합)
   {} 요소와 변경이 불가능한 집합

5. object (객체)
   데이터(자료) 기능과 결합 가능
