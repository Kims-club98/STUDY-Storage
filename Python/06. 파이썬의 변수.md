# 1. 변수

- 변수는 값을 저장하는 컨테이너(Container) 이다
- 변수는 처음 **값을 할당하는 순간** 생성된다.

```py
name="홍길동"
age=22
print("성함: ", name, "나이: ", age)
# 출력: 성함:  홍길동 나이:  22

# 변수는 특정 유형으로 설정할 필요 없으며, 설정 후에도 다른 유형(int -> str)으로 변경이 가능하다.
x=22
x="홍길동"
print(x)
#출력: 홍길동
```

# 2. Casting(형식변경)

- 변수의 데이터 형식을 지정하려면 형 변환을 사용한다.

```py
x=str(3) # 3(문자형)
y=int(3) # 3(정수형)
z=float(3) # 3.0(실수형)
f=bool(0) # False (0만) (불린)
t=bool(1) # True (0이외 전부) (불린)
print(x, y, z, f, t)
```

# 3. 해당 변수의 유형(type) 확인하기

- type()을 통해 해당 변수의 데이터의 형식을 알 수 있다.

```py
x=5
y="홍길동"
z=False
print(type(x)) # <class 'int'>
print(type(y)) # <class 'str'>
print(type(z)) # <class 'bool'>
```

# 4. 대소문자 구분

- 변수명은 대소문자를 구분한다

```py
a= 4
A="홍길동"
print(a) # 4
print(A) # 홍길동
```
