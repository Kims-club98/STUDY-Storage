# 1. 파이썬 변수 사용의 규칙

1. 변수명의 시작은 _문자_, *밑줄(\_)*로 시작해야 한다
2. 변수 명은 "숫자"로 시작할 수 없다
3. 변수명은 *0*과 _밑줄(\_)_ 만 포함될 수 있다.
4. 변수 명은 **대소문자**를 구분한다 (age ≠ Age ≠ AGE)
5. 변수명은 **python keyword**가 될 수 없다(and, as, await 등 이미 예약되어있는 명령어 집합)

```py
# 1. 유효한 변수명
myvar = "John"
my_var = "John"
_my_var = "John"
myVar = "John"
MYVAR = "John"
myvar2 = "John"

# 2. 잘못된 변수명
2myvar = "John"
my-var = "John"
my var = "John"
```

# 2. Multi Words(여러 단어로 구성된 변수 명)

- 두 단어로 이상으로 된 변수 명은 읽기 어려우므로 다음 기법을 사용한다.

## 2-1. Camel Case

    - 첫 단어를 제외한 단어를 대문자로 작성

## 2-2. Pascal Case

    - 각 단어의 첫글자를 대문자로 작성

## 2-3. Snake Case

    - 각 단어는 밑줄로 구분

```py
# Camel Case
myFirstName = "Kim"
# Pascal Case
MyFirstName = "Kim"
# Snake Case
my_first_name = "Kim"

```
