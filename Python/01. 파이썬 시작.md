# 2. 파이썬의 들여쓰기

- 다른 프로그래밍 언어의 들여쓰기는 가독성을 목적으로 하지만, 파이썬의 경우 **코드 블록**을 나타냄(매우 중요함)

```py
# if 조건 충족 시 print 함(단 print는 if문의 결과로 꼭 들여쓰기를 해야 한다.)
if 5 > 2
  print("5는 2보다 크다")

# 오류: IndentationError: expected an indented block
```

- 일반적인 들여쓰기는 4칸 (최소는 1칸)

```py
# 칸의 차이는 있으나 결과는 동일함
if 5 > 2:
 print("Five is greater than two!")
if 5 > 2:
        print("Five is greater than two!")

# 같은 코드블록에는 동일 개수의 공백이 필요 -> 이를 어길 시 오류가 발생
if 5 > 2:
 print("Five is greater than two!")
        print("Five is greater than two!")
# 오류: IndentationError: unexpected indent
```
