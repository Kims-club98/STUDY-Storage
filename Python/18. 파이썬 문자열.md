# 1. 파이썬 문자열

- 파이썬에서 문자열은 작은 따옴표, 큰따옴표로 묶는다.

# 2. 따옴표 속 따옴표 사용하기

- 문자열 내에 따옴표를 사용 가능
- 그러나 따옴표가 겹치면 안됨

```py
print("나는 그가 '그게 말이되니?' 라고 하는 것을 들었다") # 사용 가능
```

# 3. 여러 줄의 문자열

- 3개의 따옴표를 사용해 여러 줄의 변수를 할당할 수 이따.

```py
msg = '''그는 나에게 말했다.
그 일은 사실이 아니라고...
하지만 현실을 직시한 이후 그는 눈물을 참을 수 없었다. '''
print(msg)
# 그는 나에게 말했다.
# 그 일은 사실이 아니라고...
# 하지만 현실을 직시한 이후 그는 눈물을 참을 수 없었다.
```

# 4. 문자열은 배열!

- 파이썬은 유니코드 기반의 문자의 배열이다.
- 그러나 파이썬은 단일 문자는 1개의 문자열이므로 이를 배열처럼 추출할 수 있다.
- 문자의 **첫 번째는 0부터 시작한다.**

```py
a = "안녕하세요 모두!"
print(a[1]) # output: 녕
```

# 5. 문자열을 반복해서 실행하기(반복문)

- *문자열은 배열*이므로 문자열의 문자를 반복문으로 순환시킬 수 있다. for문 활용

```py
for i in "banana"
  print(i) # banana 가 1글자씩 줄 바꿈 형태로 출력(단어 1개를 1개의 객체로 인식)
```

# 6. 텍스트의 길이 구하기 len()

- 문자열의 길이를 구하기 위해서는 len()함수를 사용하면 된다.

```py
a = "Hello, World!"
print(len(a)) # 13 (공백도 1로 친다!)
```

# 7. 문자열 찾기(필터) in

- 특정 구문, 문자가 있는지 문자열(==문자배열)에서 키워드를 통해 찾을 수 있다.

```py
txt = "The best things in life are free!"
print("free" in txt) # bool 로 출력한다(True/False)
```

## 예시: 'free'가 문자열에 있는 경우만 출력하시오

```py
txt = "The best things in life are free!"
if "free" in txt:
  print("해당 문장에는 free가 존재합니다.")
```

# 8. 특정 문자가 존재하는지 확인하기 not in

- not in 을 통해 특정 문자가 문장 안에 _없는지_ 확인 가능 **(없으면 진실을 출력한다)**

```PY
txt = "The best things in life are free!"
print("expensive" not in txt) # True
```
