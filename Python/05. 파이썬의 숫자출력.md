# 1. 파이썬의 숫자 인쇄

- print()를 통해 숫자를 인쇄할 수 있다
- 하지만 숫자는 큰따옴표("", '')를 넣지 않는다

```py
# 실수,정수는 큰따옴표 없이 print 가능하다, 넣으면 오류발생
print(3)
print(358)
print(50000)

# 함수 내부에서 연산 또한 가능하다
print(3+3) # 6
print(2*3) # 6
```

# 2. 파이썬의 인쇄(문자열 + 숫자열)

- 텍스트와 숫자를 **쉼표(,)**를 기준으로 표시 가능하다.

```py
print("I am", 25, "years old.")
# 출력: I am 25 years old.
```
