# 1. 변수의 유형 지정하기(복)

- 변수의 형 변환을 통해 변수 유형을 지정할 수 있다.
- 자주 사용하는 타입: str() int() float()

```py
# int 전환
x = int(1)   # x will be 1
y = int(2.8) # y will be 2 (0.8 빠짐)
z = int("3") # z will be 3 (문자열 "3" -> 정수형 3)

# float 전환
x = float(1)     # x will be 1.0
y = float(2.8)   # y will be 2.8
z = float("3")   # z will be 3.0
w = float("4.2") # w will be 4.2

# str 전환
x = str("s1") # x will be 's1'
y = str(2)    # y will be '2'
z = str(3.0)  # z will be '3.0'
```
