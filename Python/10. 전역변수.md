# 1. 전역변수

- **함수 외부에서 생성된 변수**를 전역 변수라고 한다.
- 전역 변수는 함수 외/내부에서 누구나 사용 가능하다.

```py
x = 'hello' # 전역변수

def msg ():
  print(x + "world") # 전역변수를 함수 내부에서 사용 가능

msg() # 함수 실행
```

# 2. 지역 변수

- 함수 **내부에서 생성된 변수**는 **함수 내부에서만 사용 가능**하다

```py
x = "hello"

def msg():
  x = 'awesome'
  print(x+'world')

msg() # awesome world (함수 내에서는 지역변수가 우선)
print(x) # 지역변수를 나가면 전역변수가 다시 출력 hello
```

# 3. 지역변수 -> 전역변수 변환하기(global 키워드)

- 일반적으로 지역변수는 함수 내부에서만 사용할 수 있다.
- 그러나 함수 내부에서 전역변수를 만들고 싶다면 _global 변수명_ 으로 지역변수 -> 전역변수로 만들 수 있다.

```py
def msg()
  global x # 지역변수 -> 전역변수화
  x = 'awesome'

msg() # return값이 잆으므로 따로 출력되는 것은 없음

print('python is'+x) # python is awesome
```

- 함수 내부에서 _전역변수값을 변경_ 할때에도 global@identity를 사용한다.

```py
x = 'good'
def msg():
  global x # 지역변수로 전역변수를 변경
  x = 'fantastic'

msg() # 출력 X

print('python is' + x) # python is fantastic
```
