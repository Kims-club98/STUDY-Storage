# 1. 출력 변수

- print()는 함수 변수를 출력하는데 사용됨

## 함수 내 여러 변수를 print 할 경우 쉼표(,)로 구분한다

```py
x = 'python'
y = 'is'
z = 'good'
print(x, y, z) # python is good
print(x+y+z) # 연산자로도 여러 변수를 출력 가능함.
```

```py
x = 5
y = 10
print(x + y) # 숫자에서 연산자는 수학 연산자로 작동함 15
```

주의) 그러나 문자 + 숫자 연산자를 결합하려 +(숫자연산자) 사용 시 오류가 발생함
