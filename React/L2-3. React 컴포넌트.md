# 리엑트 컴포넌트
  1. 컴포넌트
    - 리엑트를 구성하는 최소의 단위
    - 각 컴포넌트는 자신만의 상태(state)를 가지며, 데이터를 관리하고 동적 UI를 구성함
    - 컴포넌트 == 레고

  2. 컴포넌트 구조: 클래스형과 함수형이 있으나, 현재는 함수형이 표준임

    2-1. 함수형 컴포넌트
    ```jsx
    function App(){
      return(
        <>
        출력UI
        </>
      )
    }
    ```

    2-2. Arrow function 컴포넌트
    ```jsx
    function App = () => {
      retrun(
        <>
        출력UI
        </>
      )
    }
    ```
    ### 주의: return값의 HTML의 최상의 요소는 반드시 1개여야 함!!
    2-3. 주의해햐 하는 컴포넌트 구조
    ```jsx
        function App(){
      return(
        <div>출력 UI1</div>
        <div>출력 UI2</div>
      )
    } //  ==> 오류 발생!!!!

    {/* 만약 div 요소 2개를 사용하고 싶다면??*/}
            function App(){
      return(
        <>
        <div>출력 UI1</div>
        <div>출력 UI2</div>
        </>
      )
    } //  ==> 정상작동!
    ```
   
  3. jsx란?
   - JavaScript에서 HTML과 유사하게 작성하는 방법이다.
   3-1. jsx에서 사용하는 이유
    - jsx 없이 사용 시 React.createElement() 함수를 사용해야 함
    - 그러나 jsx를 사용 시 자연스럽게 h1 사용 가능하므로 가독성에서 유리함!
   ```js
   // jsx 사용 없이 h1 사용하기
   var element = React.createElement("h1",{
    classname: "greeting"
   }, "Hello React!");
   ```
   ```jsx
   // jsx를 사용할 때
   const element = (
    <h1 className="greeting">
    Hello React!
    </h1>
   )
   ```
  3-2. jsx 사용 시 알아야 할 사항(==jsx 특징)
    1. 반드시 최상위 요소로 감싸기 (ex. <> ... </>)
    2. JS와의 충돌을 피하기 위해 다른 이름 사용
      ex.
        class -> className
        for   -> htmlfor
    3. EventHandler는 카멜케이스 형시그올 작성
      ex.
        onclick  -> onClick
        onchange -> onChange
    4. XML문법을 따름 => 여는태그 & 닫는태그 必
    5. br, input 경우에는 self_closing 으로 마무리 해야 함
      ex.
        <br />
        <input />

  4. 컴포넌트 표기방법: 자식 컴포넌트읜 有無
    - 하위 컴포넌트에 추가하기
    ```jsx
    function App() {
      return(
        <>
          <MyComponent>
            <자식 컴포넌트...>
          </MyComponent>
        </>  
      )
    }
    export default App;
    ```
    - 셀프 클로징 방식
    ```jsx
    function App() {
      return(
        <>
        <MyComponent />
        </>
      )
    }
    export default App;
    ```
  
  5.React 컴포넌트로 교체하기(HTML에서...)
    - html에서 React로 교체 방법은 HTML의 Body 내역을
    - React의 return(<> ... </>)의 ... 위치에 붙여 넣으면 된다.
  ```html
  <div> 
    <ul>
      <li>HTML</li>
      <li>REACT/li>
    </ul>
  </div>
  ```
  ____________ 변환
  ```jsx
    function App() {
      return(
        <>
        <div> 
          <ul>
            <li>HTML</li>
            <li>REACT/li>
          </ul>
        </div>
        </>
      )
    }
    export default App;
  ```
