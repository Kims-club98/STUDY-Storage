# 프롭스(Props)
  정의: 형태는 동일, 내용물만 다른 경우 동일한 형태를 반복해서 사용하기 위해 프롭스(Props)를 활용한다.
   - 프롭스는 컴포넌스 사이에 ** 객체를 전달하기 위한 객체 ** 이다.

  1. 프롭스의 특징
    - 객체이다
    - 부모 컴포넌트가 자식 컴포넌트에 ** 데이터를 전달 ** 해주기 위해 사용한다.
    - 프롭스는 ** 값, 배열, 객체, 함수 등 ** (==다양한 데이터)을 전달해줄 수 있다.
    - 프롭스는 ** 읽기전용 ** 이다. (부모 컴포넌트에서만 수정 가능하며, 자식 컴포넌트에서는 수정이 불가하다.)

  2. 프롭스 사용해보기
  ```jsx
        // 부모 컴포넌트
        function App() {
        return (
          <div>
            <h1>리액트 식당 주문 현황</h1>
            {/* 컴포넌트에 데이터를 담아 보냅니다. 이것이 Props입니다! */}
            <Food name="떡볶이" level="매움" topping="치즈" />
            <Food name="마라탕" level="보통" topping="분모자" />
          </div>
        );
      };
      // 자식 컴포넌트
      function App(props) {
        return(
          <div style={{border: '1px soild gray', margin: '10px',padding: '10px'}}>
            <h2> 메뉴: {props.name} </h2>
            <p>맴기 정도: {props.level}</p>
            <p>추가 토핑: {props.topping}</p>
          </div>
        )
      }
    export default App;
  ```
  3. 프롭스 여러 개 사용해보기
  ```jsx
  // 자식 컴포넌트
  function MyComponent (props) {
    return (  // ⚠️ 오타 수정: raturn -> return
      <>
        <h2>Props 객체 사용하기</h2>
        <p>{props.p1}</p>
        <p>{props.p2}</p>
        <p>{props.p3}</p>
        <p>{props.p4}</p>
      </>
    )
  }

  // 부모 컴포넌트
  function App () {
    return (
      <>
        <MyComponent p1={'HTML'} p2={'CSS'} p3={'React'} p4={'JavaScript'} />
      </>
    )
  }
  export default App
  ```
